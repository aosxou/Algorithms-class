# C언어 1차원 배열 예제 분석

## 프로그램 코드

```c
int main() {
    int i, j;
    for(i=0; i<100; i++)
        A[i] = i;
    j = i - 1;
    printf("%d", ㄱ);
}
```

---

## 변수 값 파악

```c
for(i=0; i<100; i++)
    A[i] = i;
// A[0]=0, A[1]=1, ... A[99]=99

j = i - 1;  // 루프 끝나면 i=100이므로
// j = 99
```

---

## 각 보기 분석

| 보기 | 분석 | 결과 |
|------|------|------|
| `A[j]` | A[99] = 99 → 출력 **100**이라 했으니 ❌ |
| `&A[0]+j` | 주소값 출력 (값 아님) ❌ |
| `*(A+j)` | *(A+99) = A[99] = **99** ✅ |
| `*A+j` | *A = A[0] = 0, 0+99 = 99 → 출력 **100**이라 했으니 ❌ |

---

## 정답

**`*(A+j)` → 출력 99**

---

## 핵심 정리

```c
A[j]   == *(A+j)   // 완전히 동일한 표현 (j번째 원소)
*A+j   == A[0]+j   // A[0]의 값에 j를 더한 것 (다른 의미!)
&A[0]+j            // 주소값이라 %d 출력에 부적합
```

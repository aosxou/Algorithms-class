# AI 기반 스마트 학습 환경 최적화 시스템

## 💡 핵심 컨셉
실내 환경을 모니터링하면서 집중 모드를 지원하는 AI 학습 도우미

---

## 🔧 하드웨어 구성 (보유 센서 활용)

### 1. **진동 센서** → 2가지 용도
- **책상 진동 감지**: 타이핑/필기 활동 감지 → 실제 공부 중인지 판단
- **외부 소음 진동**: 층간소음, 공사소음 등 학습 방해 요소 감지
  - 진동 패턴으로 소음 종류 구분 (발소리 vs 드릴 소리)

### 2. **조도 센서** → 2가지 용도
- **실내 조명 최적화**: 너무 어두우면 "조명 켜세요" 알림
- **수면 패턴 추적**: 밤에 불 꺼진 시간 → 수면 시작 추정
  - "어젯밤 3시에 잠, 오늘 집중력 저하 예상"

### 3. **초음파 센서** → 2가지 용도
- **착석 감지**: 책상 앞에 사람 있는지 감지 (거리 측정)
  - 30cm 이내 = 앉아있음, 1m 이상 = 자리 비움
- **자세 모니터링**: 거리 변화로 구부정한 자세 감지
  - "거북목 자세 20분째, 스트레칭 하세요"

### 4. **물리 버튼** (추가 필요)
- 집중 모드 시작/종료 버튼

### 5. **릴레이 모듈** (선택)
- 조명 자동 제어

## 🤖 시스템 구성

### **Hardware (아두이노)**
```
- 진동센서 → 활동량 + 외부 소음 감지
- 조도센서 → 조명 최적화 + 수면 추적
- 초음파센서 → 착석/자세 감지
- 물리 버튼 → 집중 모드 토글
- (선택) 릴레이 → 조명 제어
```

### **Backend (Python + EXAONE AI)**

#### 데이터 수집
```python
{
  "timestamp": "2026-04-28 14:30:00",
  "seated": true,              # 초음파: 착석 여부
  "distance": 25,              # 초음파: 거리(cm)
  "activity": 120,             # 진동: 활동량 (0-255)
  "noise_vibration": 45,       # 진동: 외부 소음
  "light": 350,                # 조도: lux
  "focus_mode": true,          # 버튼: 집중 모드 ON/OFF
  "session_duration": 18       # 집중 세션 경과 시간(분)
}
```

#### AI 분석 기능
1. **환경 점수 계산**
```
- 조도 적정(300-500lux): +30점
- 외부 소음 낮음: +25점
- 바른 자세 유지: +20점
- 연속 활동(타이핑): +25점
→ 총점 100점 중 현재 82점
```

2. **EXAONE 자연어 가이드**
```
입력: 센서 데이터 + 사용자 패턴
출력: "현재 조도가 250lux로 낮습니다. 
      조명을 켜면 집중력이 약 15% 향상될 것으로 예상됩니다. 
      또한 20분째 같은 자세를 유지 중이니 
      5분 후 스트레칭을 추천드립니다."
```

3. **패턴 학습**
```
- "평일 오후 2-4시, 조도 400lux일 때 집중 시간 평균 35분"
- "주말 오전, 소음 낮을 때 집중 시간 평균 50분"
→ 최적 학습 시간대/환경 제안
```

4. **이상 패턴 감지**
```
- "3일 연속 수면 시간 4시간 미만, 집중력 저하 우려"
- "오늘 착석 시간 30분, 평소의 20% 수준"
```

## 📋 집중 환경 점수 공식 기준

### 1. **조도 (Lighting) - KS 조명기준**
**출처:** 한국산업규격 KS A 3011 (조도기준)

```
독서/학습 공간 권장 조도:
- 최소: 300 lux
- 권장: 500-750 lux  ← 이걸 100점 기준으로
- 최대: 1000 lux (너무 밝으면 눈부심)

점수 계산:
if 500 <= lux <= 750: 
    score = 30점 (만점)
elif 300 <= lux < 500:
    score = 20 + (lux-300)/200 * 10  # 20-30점
elif lux < 300:
    score = max(0, lux/300 * 20)  # 0-20점
else:  # 너무 밝음
    score = 30 - (lux-750)/250 * 10  # 감점
```

### 2. **소음 (Noise) - WHO 가이드라인**
**출처:** WHO Environmental Noise Guidelines (2018)

```
학습 환경 권장 소음 수준:
- 35 dB 이하: 최적
- 35-45 dB: 양호
- 45-55 dB: 집중력 저하
- 55 dB 이상: 학습 부적합

진동센서는 dB 직접 측정 불가하므로 상대값 사용:
- 진동값 0-50: 조용함 (25점)
- 진동값 51-100: 보통 (15점)
- 진동값 101-150: 시끄러움 (5점)
- 진동값 150+: 매우 시끄러움 (0점)
```

**빠른 구현 팁:** 실제 소음계 앱으로 dB 측정 → 진동센서 값과 매핑 테이블 작성

### 3. **자세 (Posture) - 인체공학 기준**
**출처:** OSHA (미국 산업안전보건청) 사무실 인체공학

```
모니터/책 거리:
- 권장: 50-70cm
- 너무 가까움 (<40cm): 눈 피로, 거북목
- 너무 멀음 (>80cm): 집중력 저하

초음파센서 거리:
if 50 <= distance <= 70:
    score = 20점
elif 40 <= distance < 50:
    score = 15점
elif distance < 40:
    score = 10점  # 경고
else:
    score = 15점
```

### 4. **활동량 (Activity) - 실제 학습 여부**
**출처:** 자체 기준 (진동 패턴 분석)

```
진동 패턴으로 판단:
- 규칙적 진동 (타이핑/필기): 25점
- 불규칙 소폭 진동 (독서 중 움직임): 20점
- 진동 없음 30분 이상: 0점 (졸고 있음?)

간단 구현:
activity_count = 최근 1분간 진동 감지 횟수
if activity_count > 20:
    score = 25  # 활발히 공부 중
elif activity_count > 5:
    score = 20  # 독서 중
else:
    score = 0  # 활동 없음
```

## 💻 집중 환경 점수 계산 코드

```python
def calculate_focus_score(light, vibration_noise, distance, activity):
    score = 0
    
    # 1. 조도 점수 (30점) - KS 기준
    if 500 <= light <= 750:
        light_score = 30
    elif 300 <= light < 500:
        light_score = 20 + (light - 300) / 200 * 10
    elif light < 300:
        light_score = max(0, light / 300 * 20)
    else:
        light_score = max(0, 30 - (light - 750) / 250 * 10)
    
    # 2. 소음 점수 (25점) - WHO 기준 응용
    if vibration_noise < 50:
        noise_score = 25
    elif vibration_noise < 100:
        noise_score = 15
    elif vibration_noise < 150:
        noise_score = 5
    else:
        noise_score = 0
    
    # 3. 자세 점수 (20점) - OSHA 기준
    if 50 <= distance <= 70:
        posture_score = 20
    elif 40 <= distance < 50 or 70 < distance <= 80:
        posture_score = 15
    elif distance < 40:
        posture_score = 10
    else:
        posture_score = 5
    
    # 4. 활동 점수 (25점) - 자체 기준
    if activity > 20:
        activity_score = 25
    elif activity > 5:
        activity_score = 15
    else:
        activity_score = 0
    
    total_score = light_score + noise_score + posture_score + activity_score
    
    return {
        'total': round(total_score),
        'light': round(light_score),
        'noise': round(noise_score),
        'posture': round(posture_score),
        'activity': round(activity_score)
    }
```
